MIKROTIK BURST LIMIT CALCULATOR





Burst Formula:
------------------------------
BD = ( BH * BT ) / BL
------------------------------
BD = Burst Duration
BH = Burst Threshold
BT  = Burst Time
BL  = Burst Limit

*Convert 1000Kbps = 1Mbps





Upload:
BD = ( 384K * 16 ) / 1000K
      = 6 second
Burst Speed to 1000K in 6s

Download:
BD = ( 750K * 16 ) / 2000k
      = 6 second
Burst Speed to 2000K in 6s
