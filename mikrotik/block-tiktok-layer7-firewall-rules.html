Simple How To Block Tiktok Aplication Use "Layer-7" Firewall Rules RouterOS

/ip firewall layer7-protocol
add name=Tiktok regexp="^.+(tiktok.com|musical.ly).*\$"

/ip firewall filter
add action=drop chain=forward comment="BLOCK TIKTOK" layer7-protocol=Tiktok"
