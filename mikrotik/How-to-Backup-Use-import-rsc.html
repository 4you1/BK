Backup Use Import rsc - Mikrotik Script


To import the exported file is quite easy

/import file=[filename]

However, the results of imports with different routers will usually experience problems or maybe because of duplication of existing files, 
to solve this problem try using the verbose = yes command, at least by verbose we can find out where and what line the error is located.

/import file=[filename] verbose=yes
