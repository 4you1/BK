Total Queue Size:
Total Queue Size = ( Total Client * Queue Size )
Total Queue Size = ( 40 * 50 ) = 2000 total packets
40 client (sub queue) can get 50 packets per client (limit) from 2000 Total packets

PCQ Burst Result:
Burst Duration = ( Burst Threshold * Burst Time ) / Burst Rate
Burst Duration = ( 384K * 16 ) / 1000K = 6 second
Every Client (sub queue) can get Burst Speed to 1000K in 6 second
